Setting Up Your UbiquityOS Demo Environment

        
Welcome! This tool will help you set up a demo environment for UbiquityOS. Here's what will happen:

        
    
          
  1. GitHub Login: First, you'll need to log in with GitHub. We'll only request the minimum permissions (public_repo scope) needed to
            create and manage a demo repository.
    2. 
          
  2. Automatic Setup: We'll create a public demo repository in your GitHub account and configure it with all necessary UbiquityOS
            settings.
    3. 
          
  3. Interactive Demo: Install the UbiquityOS app to your repository and start the interactive demo through a GitHub issue. Our AI
            assistant will guide you through the features.
    4. 
        

        
Don't worry - everything is isolated and secure:

        
    
          
  • We use minimal GitHub permissions
    • 
          
  • The demo runs in its own repository
    • 
          
  • You can delete the repository at any time
